# Task: Project scaffold

## Description

Create the foundational project skeleton: `pyproject.toml` with deps and tooling config, Python version pin, lockfile, Makefile, `.gitignore`, package markers, and a one-paragraph README. This task blocks every other task in the epic — nothing else can begin until `uv sync` works on a fresh clone. Implements the layout side of [[2026-04-26-layered-project-structure]].

## Acceptance Criteria

- [ ] `pyproject.toml` declares `requires-python = ">=3.11,<3.12"`.
- [ ] Runtime deps in `pyproject.toml`: `fastapi`, `uvicorn[standard]`, `python-multipart`, `structlog`.
- [ ] Dev deps (in a dev/optional group): `pytest`, `pytest-cov`, `httpx`, `ruff`.
- [ ] `[tool.pytest.ini_options]` has `addopts = "--cov=app --cov-report=term-missing --cov-fail-under=95"`.
- [ ] `[tool.coverage.run]` has `source = ["app"]`.
- [ ] `[tool.ruff]` configured (line length, target version 3.11, rule selection).
- [ ] `.python-version` contains `3.11`.
- [ ] `Makefile` defines `check` (= `uv run ruff check && uv run ruff format --check && uv run pytest`) and `run` (= `uv run uvicorn app.main:app --reload`) targets.
- [ ] `.gitignore` covers `.venv/`, `__pycache__/`, `.pytest_cache/`, `.coverage`, `htmlcov/`, `.ruff_cache/`.
- [ ] Empty `__init__.py` files exist at: `app/`, `app/routers/`, `app/services/`, `app/core/`, `tests/`, `tests/unit/`, `tests/integration/`.
- [ ] `README.md` includes the fresh-clone reproducibility instruction (`uv sync && make check`) per PRD US-5.
- [ ] **Gate (mirrors Phase A):** `uv sync` succeeds on a fresh clone and `uv.lock` is committed.

## Technical Details

### Files to create

- `.python-version`

- `Makefile`
- `.gitignore`
- `README.md`
- `app/__init__.py`
- `app/routers/__init__.py`
- `app/services/__init__.py`
- `app/core/__init__.py`
- `tests/__init__.py`
- `tests/unit/__init__.py`
- `tests/integration/__init__.py`

### Notes

- The `make check` command in this task does not yet enforce coverage at the Makefile level — that's already covered by `pytest --cov-fail-under=95` in `pyproject.toml`. Don't duplicate the gate.
- Don't add any application logic in this task. Empty package markers only.

## Dependencies

- **Task deps:** none.
- **External deps:** `uv` must be installed locally to generate the lockfile.
- **ADRs implemented:** layout side of [[2026-04-26-layered-project-structure]] (creates the empty package tree).
